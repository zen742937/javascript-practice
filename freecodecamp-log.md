今天主題是 **函式（functions）與作用域**。

### 涵蓋主題

- **箭頭函式（Arrow Functions）**：它們如何運作
- **作用域（scope）**：全域、區域、區塊作用域如何運作

### 心得（zen 的筆記）
- **箭頭函式**：單行寫法（沒有 `{}`）會自動回傳、不能也不用寫 `return`（隱式回傳）；一旦包了 `{}` 變成程式區塊，要回傳值就必須自己寫 `return`，不寫會回傳 undefined。
  - 口訣：單行不寫 return（自動回傳）／要 return 就得包 `{}`。（與 React 練習 Day 24 的 onClick 同理：只有一句可省 `{}`，多句或要 return 就要 `{}`。）

## 2026-06-11｜Full Stack Developer 課程：物件存取 + 陣列方法

今天把 **物件（Object）** 的存取與操作走完，並繼續深入 **陣列（Array）** 的常用方法，最後用兩個小專案收尾。

### 物件章節
- **什麼是物件 / 如何存取屬性**（理論）— 點記法 `obj.key` 與中括號 `obj["key"]`
- **如何處理從巢狀物件和物件中的陣列存取屬性**（理論）— 一層層往下取，如 `data.user.hobbies[0]`
- **如何檢查物件是否具有某個屬性**（理論）— `"key" in obj`、`obj.hasOwnProperty("key")`
- **如何從物件中移除屬性**（理論）— `delete obj.key`

### 陣列章節
- **JavaScript 陣列小測驗 / 陣列複習**（複習 + 測驗）
- **什麼是陣列的淺層複製，以及哪些方法可以創建這些複製**（理論）— `slice()`、展開 `[...arr]`、`Array.from()`
- **如何檢查陣列是否包含特定值**（理論）— `includes()`
- **如何在陣列中間添加和移除元素**（理論）— `splice(start, deleteCount, ...items)`
- **如何使用 `indexOf` 方法取得陣列中元素的索引**（理論）— 找不到回傳 -1

### 邊學邊做的小專案
- **建置一個高爾夫球分數轉換器**
- **建置一個午餐選擇器程式**

### 心得（zen 的筆記）
- **`slice` vs `splice` 容易記混**：`slice` 不改原陣列、回傳新的（拿來做淺拷貝）；`splice` 會原地修改原陣列（拿來增刪元素）。
- **淺拷貝只拷一層**：`[...arr]`、`slice()`、`Array.from()` 對巢狀的物件/陣列仍是共用參考，改到裡層會互相影響——之後要多練「深拷貝」的時機。

今天做 **Build a Quiz Game** 實驗室，用物件陣列建一組測驗題庫，並寫三個函式讓「電腦」隨機作答、再判斷對錯。

### 完成內容
- **Build a Quiz Game**（Lab）— 全部 user stories 通過 ✅
  - `questions` 陣列：至少 5 個物件，每個含 `category` / `question` / `choices`（3 個字串）/ `answer`（須包含在 `choices` 內）
  - `getRandomQuestion(questions)` — 從題庫隨機回傳一題物件
  - `getRandomComputerChoice(choices)` — 從該題選項隨機回傳一個答案
  - `getResults(question, computerChoice)` — 答對回傳 `The computer's choice is correct!`，答錯回傳 `The computer's choice is wrong. The correct answer is: <correct-answer>`

### 心得（zen 的筆記）
